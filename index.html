Material Design

Material Design is a design system built and supported by Google designers and developers. Material.io includes in-depth UX guidance and UI component implementations for Android, Flutter, and the Web.

Using Material

The design system is organized in three main parts: foundations, styles, and components. Each section includes design guidance and tools to help you customize and build with Material.
